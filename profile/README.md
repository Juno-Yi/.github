## 介绍
JunoYi（钧逸） 是一个由年轻开发者与技术爱好者组成的现代化技术组织。
我们专注于 企业级解决方案、开源项目、开发者工具、数字化基础设施、AI 技术与生产力工具 的设计与开发。

我们相信：
技术应该是有温度的产品，也是推动世界进步的力量。

JunoYi 希望让每一位开发者、企业合作伙伴与技术热爱者
感受到技术创造价值的快乐。


## 我们的愿景

成为面向未来的技术型团队组织，用创新驱动数字世界发展。

让每一个加入 JunoYi 的人，都能在这里：
- 找到属于自己的方向
- 打造属于自己的技术力量
- 做出真正有价值、有影响力的产品
